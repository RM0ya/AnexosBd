# Anexos BD (Oracle + MongoDB)

Repositorio de anexos y evidencias del trabajo de Base de Datos, que incluye:
- Scripts SQL (Oracle): tablas/constraints, secuencias, packages y triggers.
- Evidencias de ejecución y validación (capturas y consultas).
- Modelo NoSQL (MongoDB): colecciones y documentos JSON.
- Demostración CRUD en MongoDB (comandos de creación, lectura, actualización y eliminación).

---

## Estructura del repositorio

###  Anexos
- **Anexo A – SQL Oracle Scripts**  
  Carpeta: `Anexos/A_SQL_Oracle_Scripts/`  
  Contiene: scripts de creación (tablas/constraints), secuencias, packages y triggers.

- **Anexo B – Evidencias Oracle**  
  Carpeta: `Anexos/B_Evidencias_Oracle/`  
  Contiene: capturas de `DBMS_OUTPUT`, pruebas de ejecución y validaciones con consultas `SELECT`.

- **Anexo C – Modelo NoSQL (JSON)**  
  Carpeta: `Anexos/C_NoSQL_Modelo_JSON/`  
  Contiene: exportaciones JSON de las colecciones (Alumnos, Carreras, Asignaturas, Profesores).

- **Anexo D – CRUD MongoDB**  
  Carpeta: `Anexos/D_CRUD_Mongo/`  
  Contiene: comandos CRUD (Create, Read, Update, Delete) para cada colección.


---

##  Cómo usar los anexos (rápido)
1. Revisar scripts Oracle en **Anexo A**.
2. Validar ejecución con capturas y consultas en **Anexo B**.
3. Revisar modelo documental y JSON en **Anexo C**.
4. Ejecutar CRUD desde **Anexo D**
